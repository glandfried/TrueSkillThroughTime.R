
Crear Paquetes de R. 

https://www.hsph.harvard.edu/statinformatics/soft/files/buildingrpackages.pdf

1. Cargar en un archivo todas las funciones y clases.
2. Dado 1 ejecutar package.skeleton("miPaquete")
3. Completar Descripcion
4. Completar Namesapce
5. Completar Todos los archivos .Rd
6. En terminal: R CMD build myPackageName # genera tar.gz para compartir
7. En terminal: R CMD INSTALL myPackageName 
8. En terminal: R CMD check myPackageName  

+1. R CMD Rd2pdf packName #hace un pdf a partir de los .Rd
+1. R CMD remove packName
+1. R CMD build --binary packName
